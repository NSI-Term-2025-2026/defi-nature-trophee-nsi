# Défi Nature — Trophée NSI

Projet Python réalisé dans le cadre du **Trophée NSI** : un jeu / mini-simulation autour d’un thème “nature”, avec une boucle de jeu et des interactions utilisateur.

## Objectifs
- Concevoir un programme complet (organisation, fonctions, logique de jeu)
- Travailler la conception (décomposition en fonctions, variables d’état)
- Produire un rendu utilisable et démontrable

## Prérequis
- Python 3.10+ recommandé
